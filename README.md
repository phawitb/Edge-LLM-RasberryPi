# Edge-LLM-RasberryPi

## On UbuntuPC
```
conda activate edge-llm
cd Edge-LLM-RasberryPi
chmod +x run_all.sh
./run_all.sh
```

## On RasberryPi
```
cd Edge-LLM-RasberryPi
source edge-env/bin/activate
python3 device_server.py 4 0.0.0.0 9004
```
